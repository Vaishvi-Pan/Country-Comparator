A web application that lets you compare two countries side-by-side — exploring their population, area, languages, currencies, flags, and more.

Purpose
Ever wondered how India stacks up against Brazil in population, or how many languages Switzerland speaks compared to Japan? Country Comparator makes it easy to visually compare any two countries in the world using clean, card-based layouts and real-time data.

🔗 API Used
REST Countries API

Base URL: (https://restcountries.com/v3.1/all)
No API key required — completely free and open
Provides data on 250+ countries including flags, population, area, languages, currencies, capital, and region

✨ Features

🔍 Search — Search for any country by name
⚖️ Compare — Select two countries and view them side-by-side
🗂️ Filter — Filter countries by region (Asia, Europe, Africa, Americas, Oceania)
🔃 Sort — Sort the country list by population or area (ascending/descending)
🏳️ Flag Display — See the official flag of each country
📊 Data Points — Population, area (km²), capital, region, subregion, languages, and currencies
